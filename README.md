# Mikrotik Unicode SSID
### Description
This tool lets you build wifi names with emojis.  
You can add them to your Mikrotik wifi SSID.

### Usage:
Just run the app.

### Built on:
- Visual Studio 2019

### Requires:
- Windows 10
